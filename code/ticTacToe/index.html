ticTacToe

Category: personal

ticTacToe is a simple Haskell program which I use as a recurring exercise for exploring new concepts. The core of the project is a library which I used to explore the ideas being laid out in Real World Haskellâ€™s Chapter 5 , which covers building libraries and APIs. As I progressed through the book, I explored more of the its teachings, such as adding type class instances to the library and building packages with cabal, and using the IO monad to interact with stdin and stdout.

I will continue this process as I finish the rest of RWH, so long as this project serves the concepts well. Additionally, I plan to use this API to develop a simple web app interface powered by either Snap or Yesod, whenever I know enough Haskell to understand either framework.

You can browse the code at its GitHub repo. If you want to make changes, fork it and send me a pull request. The source includes a .cabal file, so you can easily build the project with cabal configure and cabal build, or you can build and install it with cabal install.
